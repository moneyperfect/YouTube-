解决的问题

为什么要做这个

翻译 YouTube 长视频一直是个痛点。现有的工具要么需要付费订阅，要么翻译质量堪忧，要么需要把视频上传到第三方服务器。

我需要一个工具，能够在本地完成整个翻译流程，同时支持多种翻译引擎，让我可以根据需要选择最适合的。

核心能力

它能做什么

智能字幕获取

自动检测 YouTube 人工字幕、自动生成字幕，没有时回退到本地 Whisper 转写。

多引擎翻译

DeepSeek、OpenAI、通义千问、Moonshot、Ollama 本地——按需选择。

字幕编辑

单句编辑、重译、锁定、批量替换。翻译完不满意？直接改。

视频导出

一键烧录中文字幕或双语字幕到视频，支持预览和完整导出。

技术实现

怎么做的

后端 Python + FastAPI，前端原生 HTML/JS（不用框架）。视频处理用 yt-dlp 和 ffmpeg，语音转写用 faster-whisper。

整个架构设计为本地优先，所有数据处理都在用户机器上完成，不依赖第三方云服务。

Python FastAPI yt-dlp faster-whisper ffmpeg OpenAI API 原生 JS Docker

本地优先，数据不出本机。
