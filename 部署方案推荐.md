# YTSubViewer 部署方案推荐

## 按性价比排序的部署方案

### 方案一：单台 VPS 全栈部署（性价比最高）

**适合场景：** 日活 < 500 用户

一台 VPS 跑所有服务，Docker Compose 一键部署。

| 配置 | 月费 | 说明 |
|------|------|------|
| 4C8G VPS（无 GPU） | ¥50-80 | 阿里云/腾讯云轻量应用服务器 |
| 带宽 5Mbps | 含在内 | 足够并发 10-15 个下载任务 |

**关键点：**
- Whisper 转写用 CPU 跑 `distil-large-v3`，速度约为 GPU 的 1/5，但**免费**
- 如果用户主要用 YouTube 自带字幕（90%+ 的视频都有），Whisper 几乎不会被触发
- PostgreSQL + Redis 用 Docker 自建，不额外花钱

**命令：**
```bash
docker compose up -d
```

---

### 方案二：VPS + 云数据库（稳定性更好）

**适合场景：** 需要数据持久化保障、自动备份

| 组件 | 服务 | 月费 |
|------|------|------|
| Web + Worker | 4C8G VPS | ¥60 |
| PostgreSQL | 阿里云 RDS / 腾讯云 TDSQL（最低配） | ¥30-50 |
| Redis | 阿里云 Redis 1G | ¥15-30 |

**总计：¥100-140/月**

**优势：** 数据库自动备份、故障恢复、监控告警，不用操心数据丢失。

---

### 方案三：VPS + 按需 GPU（有 Whisper 高频需求）

**适合场景：** 大量无字幕视频需要转写

| 组件 | 服务 | 月费 |
|------|------|------|
| Web + Redis + PostgreSQL | 4C8G VPS | ¥60 |
| GPU Worker | AutoDL/恒源云 按需实例 | 按量付费 |

**GPU 成本参考：**
- AutoDL RTX 3090：¥1-2/小时
- 一个 1 小时视频的 Whisper 转写约 5-10 分钟
- 如果每天转写 10 个视频，月费约 ¥30-60

**架构调整：** GPU Worker 不常驻，按需启停。

---

### 方案四：海外 VPS（面向海外用户）

**适合场景：** 用户在海外，需要访问 YouTube

| 配置 | 月费 | 说明 |
|------|------|------|
| Hetzner CX31 (4C8G) | €7.5/月 ≈ ¥60 | 德国/芬兰机房 |
| DigitalOcean Droplet | $24/月 ≈ ¥175 | 美国机房 |
| Vultr | $24/月 ≈ ¥175 | 多机房可选 |

**优势：** 直接访问 YouTube，无需代理，下载速度快。

---

## 推荐总结

| 场景 | 推荐方案 | 月费 |
|------|---------|------|
| 个人项目/小团队 | 方案一：单台 VPS | ¥50-80 |
| 需要数据安全 | 方案二：VPS + 云数据库 | ¥100-140 |
| 大量无字幕视频 | 方案三：VPS + 按需 GPU | ¥90-120 |
| 面向海外用户 | 方案四：海外 VPS | ¥60-175 |

**建议：** 先用方案一（单台 VPS）跑起来，90% 的 YouTube 视频都有字幕，Whisper 基本用不到。等用户量上来再考虑拆分。
